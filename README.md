# rps
Rock Scissor Paper game 

## Installation
To install the dependencies:
```
npm install
```

## Build 
Webpack is used for the building process.
### Build to use
```
npm run build-prod
```

### Build for test
```
npm run build-test
```

## Test
The tests are based on Mocha. To run test: 
```
npm run test
```
